# APELSO 2026 個人議程表

8th Annual APELSO Conference（2026/6/11–6/13・Hamamatsucho Convention Hall, Tokyo）的個人化互動議程表。單一 HTML 檔，手機開網址即用、可離線。

## 線上使用

👉 **https://bettywilsonlin.github.io/apelso2026-agenda/**

手機瀏覽器開啟後可「加入主畫面」當 App 用。

## 功能

- **三天分頁** ＋「⭐ 我的行程」彙整
- 點卡片**展開講題**（Symposium 場全展開到逐筆講者、題目、單位）
- **加星標記**想聽的場，記在瀏覽器本機（離線保留，各裝置獨立）
- **搜尋**（講者／題目／關鍵字）、**會場篩選**、**只看已加星**
- 「我的行程」自動偵測**時間衝突**
- 會期當天標示**進行中**場次

## 資料來源與限制

| 場次 | 明細層級 | 來源 |
|---|---|---|
| 開幕式 OP、Symposium SY01–SY42 | 完整講題（題目／講者／單位） | 官網 program.html |
| Free Communications、Hands-on、Seminar | 場次層級（標題／時間／會場） | 大會時程表 PDF |
| 部分 FC／午餐會 | 手動補入的講題 | 大會 App「Conference Navi」 |

FC（含各 Doctors 場）與午餐會的逐筆講題未公開於官網，僅存於大會手機 App，採人工逐場補入，故部分場次標示「已補 N 筆」。

## 更新方式

修改 `index.html` → `git commit` → `git push`，GitHub Pages 約 1 分鐘後自動更新。
